# tex

A terminal text editor written in C. Not a professional, just trying to learn low level systems programming and building this as a project.

Still in early stages — currently handles raw terminal input and prints ASCII codes of keypresses.

## Build
```bash
make
```

## Run
```bash
./tex
```

Press `q` to quit.
